# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Instalación
1. Ejecuta `npm install` (esto instalará Vite, React y @supabase/supabase-js).

## Configuración de Entorno
1. Crea un archivo `.env` en la raíz de la carpeta frontend.
2. Agrega las variables con el prefijo obligatorio de Vite:
   VITE_SUPABASE_URL=tu_url
   VITE_SUPABASE_ANON_KEY=tu_key_anon
